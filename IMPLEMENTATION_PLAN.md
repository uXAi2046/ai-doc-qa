## 开发原则
- 每个阶段完成后进行代码审查
- 保持增量提交，确保每次提交都能编译运行
- 优先实现核心功能，后续迭代优化
- 遵循产品需求文档和技术架构文档的设计
- 重点关注Chat模块的用户体验和RAG功能的准确性

## 技术栈确认
- 前端: React 18 + TypeScript + TailwindCSS + Vite
- 后端: Node.js + Express + TypeScript
- 数据库: Supabase (PostgreSQL)
- AI服务: OpenAI API
- 部署: Docker + Vercel/Railway
