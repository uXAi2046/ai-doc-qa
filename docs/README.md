# 文档目录

本目录包含项目的相关文档和脚本文件。

## 目录结构

```
docs/
├── README.md           # 本文件，说明文档目录结构
├── scripts/            # 初始化脚本目录
│   └── init-database.md # 数据库初始化说明
└── api/                # API文档目录
    └── API.md          # API接口文档
```

## 目录说明

### scripts/
存放项目初始化相关的脚本和说明文档，包括：
- 数据库初始化脚本
- 环境配置脚本
- 部署脚本
- 其他自动化脚本

### api/
存放API接口文档，包括：
- RESTful API接口说明
- 请求/响应格式
- 认证方式
- 错误码说明
- 使用示例

## 使用说明

1. 在开始开发前，请先阅读 `scripts/init-database.md` 了解如何初始化数据库
2. 开发API时，请参考 `api/API.md` 中的接口规范
3. 新增脚本或文档时，请更新对应目录的说明文件

## 维护

请保持文档的及时更新，确保与实际代码保持同步。
